# Bundled Arabic display fonts

Used for thumbnail headlines and the subscribe-overlay button label.

| File | Family | Source |
|---|---|---|
| Lalezar-Regular.ttf | Lalezar | google/fonts `ofl/lalezar` |
| Tajawal-Bold.ttf | Tajawal | google/fonts `ofl/tajawal` |
| Cairo-Variable.ttf | Cairo (variable, wght axis) | google/fonts `ofl/cairo` |

All three are licensed under the **SIL Open Font License 1.1**
(https://scripts.sil.org/OFL). If these files are missing, the app falls back
to Windows system fonts (Arial Bold / Tahoma Bold) — functional but less
punchy. Drop any additional bold Arabic .ttf in this folder to make it
available; files whose names contain `lalezar`, `cairo`, `changa` or
`tajawal` are preferred, in that order.
